# marcosisthename.github.io

Personal resume site, built with Vue 3 + Vite. Deploys automatically to GitHub Pages on push to `main` (see `.github/workflows/deploy.yml`).

## Recommended IDE Setup

[VS Code](https://code.visualstudio.com/) + [Vue (Official)](https://marketplace.visualstudio.com/items?itemName=Vue.volar).

## Running locally

```sh
npm install   # first time only
npm run dev
```

Then open the URL it prints — note it includes the `/marcosisthename.github.io/` base path, e.g. `http://localhost:5173/marcosisthename.github.io/`. Stop the server with `Ctrl+C`.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
